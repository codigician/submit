# Submit

Submit api is the microservice that allows users to submit or run their code.

## Getting Started

Run the application
```
make run
```

Run the application in docker container
```
make up
```

Run the unit tests
```
make unit-test
```

Run all of the tests
```
make test 
```

Run code coverage
```
make code-coverage
```

Run linter
```
make lint
```
